# PontoCerto

App de controle de ponto com evidências (geolocalização + foto), aprovação de ajustes e banco de horas.

## Stack
- Mobile: React Native (Expo) + TypeScript
- API: NestJS + Prisma
- DB: PostgreSQL
- Docs: Swagger (OpenAPI)

## MVP (Semana 1)
- Login
- Bater ponto (4 eventos/dia, ordem fixa)
- Histórico por dia/mês

## Regras principais
- 4 batidas por dia (ordem automática)
- Geolocalização obrigatória
- Foto obrigatória
